05 · Research

Final Year Project

Project NAIA

Autonomous Indoor Navigation for Aerial Agents

An AI-driven drone navigation system that enables autonomous flight in GPS-denied indoor environments. Combines computer vision-based obstacle avoidance with reinforcement learning for adaptive path planning.

Supervisor: Dr. Jane SmithDepartment of Computer Science
nav_policy.pyPython
class NavigationPolicy:
    def __init__(self):
        self.encoder = ResNet18(pretrained=True)
        self.policy = PPO(actor_dim=512, action_dim=4)
    
    def select_action(self, obs):
        features = self.encoder(obs)
        return self.policy(features)
  • 95.3% obstacle avoidance accuracy in simulation
  • Real-time inference at 30 FPS on Jetson Nano
  • Zero-shot transfer to physical Crazyflie drone
